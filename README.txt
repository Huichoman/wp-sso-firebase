== Description ==

This plugin let's you authenticate with Firebase. Once authenticated, a user account will be created if does not exists.

= Features =
*   **Firebase Single Sign-on Authentication**
    a) Email/Password
    b) Facebook
    c) Google
*   **Account Creation** : once authenticated, WordPress is created.

== Installation ==

From your WordPress dashboard

1. **Visit** Plugins > Add New
2. **Search** for "Firebase Auth Sign-in"
3. **Activate** Firebase Auth Sign-in from your Plugins page
4. **Click** on the new menu item "WP Firebase"
5. **Fill-in** your API Key and Authorized Domain and save config
6. **Check/Uncheck** Sign-in providers
