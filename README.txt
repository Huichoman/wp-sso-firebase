== Description ==

This plugin let's you authenticate with Firebase. Once authenticated, a user account will be created if does not exists.

= Features =
*   **Firebase Single Sign-on Authentication**
    a) Email/Password
    b) Facebook
    c) Google
*   **Account Creation** : once authenticated, WordPress is created.
